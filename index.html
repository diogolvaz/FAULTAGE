Introduction

Fault tolerant distributed algorithms are essential on the support of modern distributed systems by preserving the regular operation of such systems even when some of the nodes fail. However, the development of such algorithms is a complex, time-consuming, error-prone and manual task.

This project aims to revolutionize the field: we propose an approach that automatically generates fault tolerant distributed algorithms from a high-level specification using machine learning techniques. With the emerge of Generative AI tools, such as ChatGPT and GitHub Co-Pilot, we aim to use the same logic on the field of distributed algorithms. Currently, we have been able to successfully generate fault-tolerant Reliable Broadcast algorithms and are now focused on the Consensus problem (check Algorithms section).
